The street network images—in the style of Nolli maps—are based off those hand-drawn by Allan Jacobs in his classic book on street-level urban form and design, Great Steets. Geoff Boeing, urban planning and spatial analysis professor at USC, developed Python code using the OSMnx package to recreate these maps for any city in the world. You can find the code to reproduce the diagrams in his GitHub Repository.

A few existing JSFiddles were used as examples for this project, mainly to set up the image overlays and to create the outcome button.
